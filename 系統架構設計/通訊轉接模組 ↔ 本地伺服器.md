通訊轉接模組 ↔ 本地伺服器
---

1. 標準 json 格式: Cmd, {Server,} {Module,} Device, Function, Value, {Auth-Token}
範例: 設定空調温度為 26 度。(由 Server 發向通訊轉接模組)
````
{
    "Cmd": 1,
    "Module": "Your Module ID",
    "Device": "Conditioner",
    "Function": "Temperature",
    "Value": 26
}
````

2. 第一次以 RPC 方式向管理系統註冊:
    * 送出本模組所擁有的 "設備/功能/類型/值域/說明/UI 元件":
    ````
    {
        "Cmd": 99,
        "Module": "",
        "Devices": {
            "Device1": {
                "DI-1": {
                    
                }
            },
            "Device2": {

            }
        }
    }
    ````
